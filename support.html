Kamuriki Linux 2.4.2以前をお使いの方へ(2023/07/12)

重要なお知らせですのでご一読ください。

OSDNのネットワーク障害によるダウンロードの不具合について(2023/07/12)

現在OSDNにて接続障害が発生しています。原因は公式からのアナウンスがないため不明です。
また、これに伴いSourceForgeへのドキュメントなど一部コンテンツの移行作業が止まっています。
リポジトリについては移行済みであるため、ユーザーの皆様にはお手数をお掛けしますが、以下の作業を行なって下さい。

●Univalentをお使いの方
以下のコマンドを実行して、キーリング及びミラーリストを更新して下さい。

wget https/sourceforge.net/projects/univalentgnulinux/files/repo/univalent-stable/x86_64/univalent-keyring-20230722-2-any.pkg.tar.zst sudo pacman -U univalent-keyring-20239722-2-any.pkg.tar.zst sudo pacman -U "https://sourceforge.net/projects/univalentgnulinux/files/repo/univalent-stable/x86_64/univalent-mirrorlist-20230710-1-any.pkg.tar.zst"
sudo pacman -Syyu
●Kamurikiをお使いの方
リポジトリ情報はbase-fileskamuriki:mainに含まれるので、以下のコマンドで更新をして下さい。

・1.xをお使いの方
wget https://sourceforge.net/projects/kamurikilinux/files/repo/kamuriki/pool/main/b/base-files/base-files_11ubuntu2023.07.22kamuriki1.8.2_amd64.deb
sudo apt install ./base-files_11ubuntu2023.07.22kamuriki1.8.2_amd64.deb
・2.xをお使いの方
wget https://sourceforge.net/projects/kamurikilinux/files/repo/kamuriki/pool/main/b/base-files/base-files_12ubuntu2023.07.22kamuriki2.4.2_amd64.deb
sudo apt install ./base-files_12ubuntu2023.07.22kamuriki2.4.2_amd64.deb
